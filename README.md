# Tic Tac Toe (Python + Tkinter)

An interactive desktop Tic-Tac-Toe game built using Python and Tkinter.

##  Features
- Separate backend (`logic.py`) and frontend (`app.py`)
- 3x3 interactive GUI grid
- Win and draw detection
- Restart button
- Two-player mode (X vs O)
